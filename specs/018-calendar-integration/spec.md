# 018: Calendar Integration & Scheduling

**Status:** Draft
**Epic:** Productivity & Organization
**Priority:** P2
**Assigned To:** TBD
**Target Release:** v1.2

## 1. Overview

Calendar integration for scheduling therapy sessions, parent meetings, and task deadlines. Syncs with Google Calendar, Outlook, and Apple Calendar.

## 2. Key Features

- Session scheduler with availability slots
- Automated reminders (email, push notifications)
- Recurring appointments
- Conflict detection
- Task deadline visualization
- 2-way sync with external calendars

## 3. Appointment Types

- Therapy sessions (1-on-1)
- Parent-teacher conferences
- Group sessions
- IEP meetings
- Progress review meetings
- Task deadlines

## 4. Technical Requirements

- Google Calendar API integration
- Microsoft Graph API (Outlook)
- iCal format export
- Timezone handling
- Reminder service (cron jobs)
