# 022: Voice Notes & Audio Feedback

**Status:** Draft
**Epic:** Communication
**Priority:** P2
**Assigned To:** TBD
**Target Release:** v1.2

## 1. Overview

Voice recording for notes and feedback instead of typing. Especially useful for therapists during sessions and students with writing difficulties.

## 2. Key Features

- Record audio notes (up to 5 minutes)
- Auto-transcription using Whisper API
- Attach to student profiles, tasks, adaptations
- Playback speed control
- Download as MP3
- Search transcriptions

## 3. Use Cases

- Therapist session notes (speak instead of type)
- Student task explanations
- Parent voice messages to therapist
- Audio feedback on completed tasks
- Student self-reflection recordings

## 4. Technical Requirements

- Web Audio API for recording
- OpenAI Whisper for transcription
- S3 storage for audio files
- Compression (opus codec)
- Waveform visualization
